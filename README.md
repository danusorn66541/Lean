# ระบบบันทึกการใช้เวชภัณฑ์ด้วย QR Code

## โครงการ Lean

### โครงสร้างไฟล์
- index.html - หน้าหลัก
- styles.css - สไตล์
- app.js - ตรรกะหลักของระบบ
- data/ - โฟลเดอร์ข้อมูล (สามารถเพิ่มได้)

### วิธีรัน
1. เปิดโฟลเดอร์ใน VS Code
2. ใช้ Live Server (extension) หรือเปิด index.html โดยตรง
3. พัฒนาต่อได้เลย

ระบบนี้พร้อมสำหรับการพัฒนาต่อ (เพิ่ม Backend, Database, Real QR Scanner ด้วย camera API)
